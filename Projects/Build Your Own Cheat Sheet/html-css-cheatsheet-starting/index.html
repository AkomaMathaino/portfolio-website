HTML Reference

Tags

Tag Name Description
<!DOCTYPE html> HTML Document Type Defines the document type
<html> HTML Defines the root of an HTML document
<link> Link Defines the relationship between a document and an external resource
<button> Button Defines a clickable button
<a> Anchor Defines a hyperlink
<img> Image Defines an image
<video> Video Defines embedded video content
<audio> Audio Defines embedded sound content
<nav> Navigation Defines navigation links
<header> Header Defines a header for a document or section
<article> Article Defines an article
<main> Main Specifies the main content of a document
<body> Body Defines the document's body
<aside> Aside Defines content aside from the page content
<footer> Footer Defines a footer for a document or section
<h1> to <h6> Headings Defines HTML headings
<p> Paragraph Defines a paragraph
<div> Divider Defines a section in a document
<section> Section Defines a section in a document
<span> Span Defines a section in a document
<table> Table Defines a table
<thead> Table Head Groups the header content in a table
<tbody> Table Body Groups the body content in a table
<tfoot> Table Foot Groups the footer content in a table
<tr> Table Row Defines a row in a table
<th> Table Header Defines a header cell in a table
<td> Table Data Defines a cell in a table
